## Comment exécuter le projet

### Prérequis
- CMake 3.10+
- Un système Linux

### Installation
1. **Cloner le dépôt**
```bash
git clone https://github.com/loicpgt/http.git
```
2. **Générer le projet**
```bash
mkdir bin
cd bin
cmake ..
cmake --build .
```
